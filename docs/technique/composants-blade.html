Principe d'utilisation

Les composants métier acceptent des props pour être configurables sans duplication. Exemples :

<x-page-header title="Mes biens" :count="$properties->count()">
    <x-slot:action>
        <a href="{{ route('properties.create') }}">Ajouter</a>
    </x-slot:action>
</x-page-header>

<x-status-badge :status="$property->status" />

<x-stat-card label="Biens occupés" :value="$occupied" icon="home" color="green" />

<x-progress-bar :value="$maintenance->progress_percentage" />
Pour ajouter un composant : créer le fichier dans resources/views/components/. Laravel le détecte automatiquement via la convention de nommage — aucune déclaration manuelle requise.
