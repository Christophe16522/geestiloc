Installation

Setup de l'application Laravel pas à pas.

Prérequis

  • PHP 8.1+
  • Composer
  • Node.js 18+ et npm
  • SQLite (inclus dans PHP) ou MySQL

1. Cloner le dépôt

git clone <url-du-repo> gestiloc
cd gestiloc/projet-dynamic

2. Installer les dépendances PHP

composer install

3. Configurer l'environnement

cp .env.example .env
php artisan key:generate

Éditer .env selon votre environnement :

# SQLite (développement)
DB_CONNECTION=sqlite
# Le fichier sera créé à database/database.sqlite

# MySQL (production)
# DB_CONNECTION=mysql
# DB_HOST=127.0.0.1
# DB_PORT=3306
# DB_DATABASE=gestiloc
# DB_USERNAME=root
# DB_PASSWORD=secret

4. Créer la base de données SQLite

touch database/database.sqlite

Cette étape est uniquement nécessaire pour SQLite. Sur MySQL, créer la base via votre interface habituelle.

5. Exécuter les migrations

php artisan migrate

Optionnel — peupler avec des données de test :

php artisan db:seed

6. Installer les dépendances front-end

npm install

7. Compiler les assets

# Développement (avec hot reload)
npm run dev

# Production
npm run build

8. Lancer le serveur

php artisan serve

L'application est accessible sur http://localhost:8000.

Récapitulatif rapide

composer install
cp .env.example .env
php artisan key:generate
touch database/database.sqlite
php artisan migrate
npm install && npm run dev
php artisan serve
Pour le développement, laissez npm run dev tourner dans un terminal séparé afin de bénéficier du hot module replacement (HMR) de Vite.
