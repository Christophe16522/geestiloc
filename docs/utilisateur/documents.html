Documents

Centraliser et retrouver tous les fichiers liés à vos biens.

À quoi sert ce module ?

Le module Documents vous permet de stocker en ligne tous les fichiers liés à votre activité de bailleur : contrats de bail signés, diagnostics obligatoires (DPE, électricité…), quittances de loyer, attestations d'assurance, et tout autre document utile.

Uploader un document

  1. Cliquez sur Documents dans le menu latéral.
  2. Cliquez sur Ajouter un document.
  3. Remplissez le formulaire :
ChampDescriptionObligatoire
NomNom du document (ex : « DPE appartement Lyon »)Oui
CatégorieContrat, diagnostic, quittance, assurance ou autreOui
Bien associéBien auquel ce document se rattacheNon
Date d'uploadDate de dépôt du documentOui
Date d'expirationDate de validité (ex : pour un DPE ou une assurance)Non
FichierLe fichier à uploader (PDF, image, etc.)Oui
  1. Cliquez sur Uploader.

Catégories de documents

CatégorieExemples
ContratBail de location signé, avenant au bail
DiagnosticDPE, diagnostic électricité, amiante, plomb, gaz
QuittanceQuittance de loyer mensuelle
AssuranceAttestation assurance propriétaire non occupant (PNO), attestation locataire
AutrePhotos, relevés de charges, factures de travaux

Télécharger un document

Dans la liste des documents, cliquez sur le nom ou le bouton Télécharger pour récupérer le fichier sur votre appareil.

Supprimer un document

Cliquez sur Supprimer dans la liste. La suppression est définitive. Si vous avez uploadé un mauvais fichier, supprimez-le et ré-uploadez le bon.

Il n'est pas possible de modifier un document déjà uploadé. Pour remplacer un fichier, supprimez l'ancienne version puis uploadez la nouvelle.

Bonne pratique : les diagnostics

Renseignez la date d'expiration pour vos diagnostics obligatoires (DPE valable 10 ans, diagnostic électricité 6 ans, etc.). Cela vous permettra de suivre leur validité et d'anticiper leur renouvellement.
