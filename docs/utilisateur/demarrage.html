Premiers pas

Créer votre compte, vous connecter et configurer votre profil.

Créer un compte

Si vous n'avez pas encore de compte, rendez-vous sur la page d'inscription accessible depuis la page de connexion.

  1. Cliquez sur S'inscrire depuis la page de connexion.
  2. Renseignez votre nom, votre adresse email et un mot de passe.
  3. Cliquez sur S'inscrire pour valider.
  4. Vous êtes automatiquement connecté et redirigé vers le tableau de bord.
Chaque compte est indépendant : vos biens, locataires et données ne sont visibles que par vous. Plusieurs bailleurs peuvent utiliser la même application sans se voir.

Se connecter

  1. Accédez à l'application dans votre navigateur.
  2. Saisissez votre adresse email et votre mot de passe.
  3. Cliquez sur Se connecter.

Si vous avez oublié votre mot de passe, utilisez le lien Mot de passe oublié ? présent sur la page de connexion.

Se déconnecter

Cliquez sur votre nom en haut à droite de la navigation, puis sur Se déconnecter.

Modifier votre profil

Pour changer votre nom, votre adresse email ou votre mot de passe :

  1. Cliquez sur votre nom en haut à droite.
  2. Sélectionnez Profil.
  3. Modifiez les informations souhaitées.
  4. Cliquez sur Enregistrer.

Supprimer votre compte

La suppression de votre compte est définitive. Toutes vos données (biens, locataires, contrats, paiements, documents, maintenances) seront effacées sans possibilité de récupération.

Cette option se trouve en bas de la page Profil, dans la section Supprimer le compte. Une confirmation par mot de passe est requise.

Étape suivante

Une fois connecté, commencez par ajouter vos biens immobiliers. Consultez la section Mes biens pour savoir comment procéder.
