GestiLoc Docs

Module : Propriétés

Gestion du parc immobilier du bailleur.

Description

Chaque bien appartient à un utilisateur (user_id) et peut être de types variés. Une référence unique est auto-générée à la création (ex : PROP-001).

Champs

ChampTypeDescription
referencestring uniqueAuto-générée (ex : PROP-001)
namestringNom / libellé du bien
addressstringAdresse
citystringVille
postal_codestringCode postal (nullable)
typeenumappartement, maison, studio, commercial, autre
surface_m2decimalSurface en m² (nullable)
monthly_rentdecimalLoyer mensuel
chargesdecimalCharges mensuelles (défaut : 0)
depositdecimalDépôt de garantie (défaut : 0)
statusenumoccupee, vacante
descriptiontextDescription libre (nullable)

Relations

  • Appartient à un User
  • A plusieurs Tenant, Contract, Payment, Document, Maintenance

La suppression d'un bien déclenche une suppression en cascade de toutes ses données associées.

Accessors Eloquent

  • full_address — adresse complète formatée (adresse + code postal + ville)
  • occupancy_rate — taux d'occupation (100% si occupé, 0% si vacant)
