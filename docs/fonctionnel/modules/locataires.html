Module : Locataires

Gestion des locataires associés aux biens immobiliers.

Description

Chaque locataire appartient à un utilisateur et peut être rattaché à un bien (property_id nullable). Le statut de paiement est mis à jour manuellement ou via les actions du module Paiements.

Champs

ChampTypeDescription
first_namestringPrénom
last_namestringNom
emailstringEmail (nullable)
phonestringTéléphone (nullable)
property_idFKBien associé (nullable — mis à null si bien supprimé)
monthly_rentdecimalLoyer mensuel (défaut : 0)
lease_start_datedateDate de début de bail (nullable)
payment_statusenumpaye, attente, retard

Accessors Eloquent

  • full_name — prénom + nom concaténés
  • initials — initiales en majuscules (ex : JD pour Jean Dupont)
  • is_late — booléen, vrai si payment_status === 'retard'

Relations

  • Appartient à un User
  • Appartient à une Property (nullable)
  • A plusieurs Payment, Contract

Scopes Eloquent

  • scopePaye — filtre les locataires à jour
  • scopeAttente — filtre les locataires en attente de paiement
  • scopeRetard — filtre les locataires en retard
  • scopeSearch($search) — recherche par prénom, nom ou email
