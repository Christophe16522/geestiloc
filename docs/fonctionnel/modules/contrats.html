GestiLoc Docs

Module : Contrats

Gestion des baux de location, archivage et alertes d'expiration.

Description

Un contrat lie un locataire à un bien pour une période donnée. La date de fin est nullable pour les baux à durée indéterminée.

Champs

ChampTypeDescription
tenant_idFKLocataire (cascade delete)
property_idFKBien (cascade delete)
typeenumvide, meuble, commercial
start_datedateDate de début
end_datedateDate de fin (nullable — bail illimité)
monthly_rentdecimalLoyer mensuel
chargesdecimalCharges (défaut : 0)
depositdecimalDépôt de garantie (défaut : 0)
statusenumactif, expire, archive

Accessors Eloquent

  • duration — durée en mois (null si pas de date de fin)
  • is_expiring — vrai si le contrat expire dans les 30 jours

Scopes Eloquent

  • scopeActif — filtre les contrats actifs
  • scopeExpirant($days = 30) — contrats actifs expirant dans N jours
  • scopeArchive — filtre les contrats archivés

Actions spéciales

MéthodeURLDescription
PATCH/contracts/{id}/archivePasse le statut à archive

Relations

  • Appartient à un User, un Tenant, une Property
