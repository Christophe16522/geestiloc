Module : Documents

Upload et gestion des fichiers liés aux biens immobiliers.

Description

Centralise les fichiers liés aux biens : contrats scannés, diagnostics obligatoires, quittances de loyer, attestations d'assurance, etc. Chaque document est rattaché à un bien (property_id nullable).

Champs

ChampTypeDescription
property_idFKBien associé (nullable — mis à null si bien supprimé)
namestringNom du document
categoryenumcontrat, diagnostic, quittance, assurance, autre
upload_datedateDate d'upload
expiration_datedateDate d'expiration (nullable)
file_pathstringChemin du fichier stocké (storage)
file_sizebigintTaille en octets (défaut : 0)
mime_typestringType MIME (nullable)

Relations

  • Appartient à un User
  • Appartient à une Property (nullable)
Les actions edit et update sont désactivées pour ce module (->except(['edit', 'update']) dans les routes). Pour modifier un document, il faut le supprimer et le ré-uploader.
